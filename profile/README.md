# hunnu-labs

A collection of projects exploring programming languages, compilers, and developer tools.

## Projects

| Project | Description |
|---------|-------------|
| [hunnu-lang](https://github.com/hunnu-labs/hunnu-lang) | A lightweight, expression-oriented programming language with C interpreter, Rust AOT compiler, OOP, FFI, generics, and Mongolian i18n |
| [hunnu-labs-landing](https://github.com/hunnu-labs/hunnu-labs-landing) | Organization landing page (React + Vite + NestJS monorepo) |
